1) выгрузка:
pg_dump -U postgres -d <имя базы данных> -f <полный путь к файлу>

2) добавить строчки после первого блока set:
DROP DATABASE <имя базы данных>;
--
-- Name: <имя базы данных>; Type: DATABASE; Schema: -; Owner: -
--

CREATE DATABASE <имя базы данных>;


\connect <имя базы данных>

3) восстановление:
psql -U postgres -f <полный путь к файлу>


!!!! 
1) <имя базы данных> должно быть маленькими буквами
2) база данных должна быть закрыта
