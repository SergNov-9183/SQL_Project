Функции:
	1) add_district (<название района>)
		SELECT add_district ('Кстовский');

	2) add_settlement (<название района>, <название нас.пункта (село/город)>)
		SELECT add_settlement('Чкаловский', 'село','Катунки');

	3) add_street (<название нас.пункта>, <название улицы>)
		SELECT add_street ('Пурех', 'Ровная')

	4) add_house (<название улицы>, <тип дома (private/multiroom)>, <номер дома>, <номер корпуса>, <площадь участка>)
		SELECT add_house('Прекрасная', 'multiroom', '75', '2', Null)
		SELECT add_house('Центральная', 'private', '12', Null, 45)

	5) add_realtor_firm (<id_дома>, <названия компании>, <рейтинг>, <телефон>, <эл.почта>, <Описание>)
		SELECT add_realtor_firm (16, 'TopTopCompany', 4.3, '297-89-22', 'top_realtors.here@inbox.ru', 'Top of the top company ever!');

	6) add_human (<фамилия>, <имя>, <отчество>, <телефон>, <эл.почта>, <пароль>, <название риэлторской фирмы>)
		SELECT add_human('Грибоедова', 'Наталья', 'Ивановна', '89038652995', 'hey.natasha@gmail.com', 'Is_42_The_Answer?', 'TopTopCompany');

	7) add_advert (<id человека>, <id дома>, <название нас.пункта>, <номер квартиры>, <кол-во комнат>, <общая площадь>, <жилая площадь>, <площадь кухни>, <водопровод (True/False)>, <газ (True/False)>, <электричество (True/False)>, <канализация (True/False)>, <тип санузла (совмещённый/раздельный)>, <категория (покупка/продажа/сдача/аренда)>, <Текст объявления>, <цена>, <дата публикации (в одном из форматов timestamp without timezone в Postgres. Например, '01-01-2010 05:06:48')>, <Доп. информация>.


	8) get_login_data ()
		SELECT get_login_data();





PS:
1) Если аргумент пустой, передаётся Null
2) Там, где принимается текст, аргументы в взять кавычки
3) Если человек не состоит в риэлторской фирме, название его риэлторской фирмы 'zero'
4) zero - всё, что связано с несуществующими объектами
